# DEPLOY DEPENDENCIES

1. Docker
2. Docker compose

# DEPLOY

1. Clone repository
2. docker-compose build && docker-compose up -d
3. Servicees can be discover on tcp-ports:
  - 82 - Apache
  - 83 - goahead
  - 84 - Nginx

# USAGE

Put some web-data to ```/var/www/``` directory on host
